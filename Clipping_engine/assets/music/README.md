# Background Music Assets

Place royalty-free `.mp3` files here. Each file must match a `"file"` entry in `catalog.json`.

## Recommended free sources

- [Pixabay Music](https://pixabay.com/music/) — no attribution required
- [Free Music Archive](https://freemusicarchive.org/)
- [Uppbeat](https://uppbeat.io/) — free tier with attribution

## Expected files

```
chill_lofi.mp3
upbeat_pop.mp3
cinematic_epic.mp3
energetic_edm.mp3
acoustic_chill.mp3
hip_hop_beat.mp3
dramatic_tension.mp3
happy_ukulele.mp3
```

Download tracks, rename to match the filenames above, and drop them in this folder.
